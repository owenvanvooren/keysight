# keysight
keysight is a super simple utility for macOS (specifically the backlightless MacBook Neo) that illuminates your keyboard with the help of a bright white bar at the bottom of your display. it intelligently detects when your cursor is near it, and puts a halo around it so you can see what you're clicking, letting you fully interact with UI elements like the Dock. (inspiration from FaceTime Edge Light)

to build, simply download the project, unzip the file, and open in Xcode, then enter your free/paid developer credentials and build!
